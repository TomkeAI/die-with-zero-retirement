# 清远 2 天 1 夜 · 溯溪登山玩水吃鸡攻略

**主题：** 不漂流，低价门票，溯溪登山玩水，清远鸡吃到爽

---

## 行前准备

| 物品 | 说明 |
|------|------|
| 溯溪鞋 | 最重要，溪底石头滑，拖鞋禁止 |
| 防水包 + 手机防水袋 | 玩水必备 |
| 速干衣裤 + 换洗衣服 | 湿身是必然的 |
| 驱蚊水 + 防晒霜 | 山里蚊虫多 |
| 少量干粮和水 | 部分地点补给有限 |

---

## Day 1 · 太和古洞溯溪 + 清城吃鸡

### 上午 10:00 — 广州/深圳出发 → 清远

自驾广清高速约 1.5-2h。高铁到清远站后建议租车或打车。

### 中午 12:00 — 午餐：太和古洞附近柴火鸡

太和古洞周边很多农家菜馆，必点**柴火鸡**（农家灶台柴火烧制，锅边贴饼子吸汁），吃饱再进山。

### 下午 1:00 — 太和古洞溯溪玩水

| 项目 | 内容 |
|------|------|
| 地址 | 清新区太和镇，距市区仅 5 公里 |
| 导航 | 太和古洞风景区 |
| 门票 | **¥15/人**（含山泉水泳池 + 溯溪 + 登山，学生半价） |
| 开放 | 7:00-18:00 |
| 停车 | 景区停车场 |
| 难度 | ★☆☆☆☆ 轻量级，亲子友好 |

**进景区后的游玩顺序（从下往上）：**

**第一站：山泉水泳池（入口附近）**
标准泳池，水是从山上直接引下来的山泉水活水，比普通泳池凉快很多、水质更干净。检票进门就能看到，建议先游一圈热身。

**第二站：浅滩溯溪（沿溪而上 1.5km）**
泳池往上走就是溪流浅滩，水不深，多处小水潭。找个人少的地方下水泡着，水清凉透彻，夏天非常舒服。

**第三站：太和古洞（最上游）**
走到尽头是道教古洞遗址，有石刻、古庙，山间阴凉。不赶时间可以上去看看。

> **推荐节奏：** 进景区 → 先游山泉水泳池 → 沿溪溯溪玩水 → 到太和古洞看看 → 下山再游一次泳池 → 出去吃柴火鸡

### 下午 5:00 — 回清城市区入住

太和古洞离市区 5 分钟车程。推荐住**清城区**（方便晚上扫街吃鸡）：

| 档次 | 推荐 | 价格 |
|------|------|------|
| 经济 | 城市便捷 / 7 天连锁 | ¥150-250 |
| 舒适 | 希尔顿欢朋 / 清远国际酒店 | ¥300-500 |

### 下午 5:30 — 清城寻鸡（重点）

清城区遍地清远鸡专门店，按吃法推荐：

| 吃法 | 推荐理由 | 找店关键词 |
|------|---------|-----------|
| **桑拿鸡** | 蒸汽锁鲜，嫩到入口即化 | 老城记脆皮桑拿鸡 |
| **五指毛桃鸡煲** | 椰香回甘，药膳汤底 | 莫氏鸡煲（排队久）/ 街边瓦煲鸡店 |
| **吊烧鸡** | 外脆里嫩，越嚼越香 | 大燚果木烧鹅烧鸡 |
| **猪肚鸡** | 浓汤暖心暖胃 | 猪肚鸡专门店 |
| **白切鸡** | 经典原味，皮爽肉滑 | 任何本地粤菜馆 |
| **铜盘鸡** | 红葱头生姜焗制，满屋飘香 | 清城粤菜小馆 |

人均 ¥50-80 能吃很好。不想排队的直接搜"大燚果木烧鹅"或"老城记脆皮桑拿鸡"，品质有保障。

### 晚上 8:00 — 飞来湖公园散步（免费）

夜景不错，适合消食散步。

## Day 2 · 南峡徒步 + 返程

### 上午 8:00 — 早餐 & 退房

来一碗清远特色**牛腩捞面**或**沙茶牛肉捞面**，¥10-15。

### 上午 9:00 — 南峡森林公园徒步

| 项目 | 内容 |
|------|------|
| 地址 | 清城区飞来峡镇 |
| 导航 | **南峡森林公园** |
| 停车 | ¥10（开进去停公园里面） |
| 门票 | **免费**（入口扫码登记） |
| 登顶 | 约 1 小时 |
| 环线 | 约 10km / 爬升 600m |
| 山顶标志 | "今日冠"大石，正对北江 |
| 补给 | 半山有小卖部 |

**两条路线可选：**

| 路线 | 耗时 | 适合 |
|------|------|------|
| 楼梯直上（经典） | 1h 上 + 40min 下 | 有体力、想快速登顶看江景 |
| 大马路（南峡山农场方向） | 1.5h 上，水泥路平缓 | 膝盖不好或轻松走 |

**亮点：** 山顶视野是清远徒步路线里数一数二的——脚下是北江蜿蜒，远处是飞来峡大坝，山水画卷尽收眼底。

### 上午 11:00 — 下山，午饭

下山后找一家做**铜盘鸡**或**瓦煲鸡**的店，吃完有力气开车。

### 下午 1:00 — 返程前加餐（可选）

如果时间充裕，绕去**清远大石板**免费玩水点泡个脚再走。

## 安全须知

- ⚠ 暴雨后、涨水期**严禁**进入溪谷
- ⚠ 不要盲目跳水，水下可能有暗石
- ⚠ 深水区必须穿救生衣
- ✅ 穿溯溪鞋，不穿拖鞋
- ✅ 出发前查天气预报
- ✅ 带走所有垃圾，无痕户外
